# Development Startup Checklist

- [ ] Terminal works
  - Run `pwd`
  - Run `ls`

- [ ] Node.js is available
  - Run `node -v`

- [ ] npm and npx are available
  - Run `npm -v`
  - Run `npx --version`

- [ ] Git is available
  - Run `git --version`

- [ ] Git user settings are configured
  - Run `git config --global user.name`
  - Run `git config --global user.email`

- [ ] Working directory is correct
  - Run `pwd`
  - Confirm the current folder is the intended project folder

- [ ] Repository state is clean or understood
  - Run `git status`
  - Confirm uncommitted changes are expected

- [ ] Required project files exist
  - Check for `package.json`
  - Check for `.env`
  - Check for `README`

- [ ] Dependencies can be installed
  - Run `npm install` if needed

- [ ] App or dev server starts
  - Run `npm start` or `npm run dev`

- [ ] Basic Node.js execution works
  - Run `node -e "console.log('ok')"`

- [ ] Codex can edit files
  - Create or edit a small test file

- [ ] Git can save work
  - Run `git add .`
  - Run `git commit -m "chore: initial setup check"`
